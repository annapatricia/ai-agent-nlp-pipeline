# AI Agent + NLP Extraction Pipeline

Protótipo de agente de IA para estruturar dados não estruturados e extrair informações com técnicas de PLN.
Inclui pipeline de ingestão, pré-processamento, extração e geração de relatórios.

## Estrutura
- src/ - código do agente e pipeline
- docs/ - documentação técnica
- notebooks/ - demonstrações
- outputs/ - saídas geradas (ex.: JSON/CSV)
